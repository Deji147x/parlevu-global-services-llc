## Step-by-Step Posting (Buffer.com)

```
Buffer.com → Compose (blue button, top right)
```

### 2. Select Platforms
Check which platforms to post to:
- ☑ Facebook
- ☑ Instagram  
- ☑ LinkedIn
- ☑ X (Twitter)
- ☑ Pinterest

*Tip: Different platforms benefit from different captions. See Platform-Specific Captions below.*

### 3. Enter Caption
```
Copy from BUFFER_CAPTIONS.txt for the blog post you're sharing
```

Example caption:
```
✨ We Buy Houses Baltimore Fast — No Realtor, No Wait, No Repairs

Your Baltimore home can close in a week. No agent. No repairs. We handle it.

Learn more about selling your Baltimore home for cash. No repairs. No fees. No agent commissions.
```

### 4. Add Link
```
Paste blog URL in the "Link" or "Add Link" field:
https://www.parlevugloballlc.com/blog-we-buy-houses-baltimore-fast.html
```

### 5. Schedule Post
```
Click "Schedule" (not "Post Now")
Select date: Monday, Wednesday, or Friday
Select time: 9:00 AM (your timezone)
Click "Save"
```

### 6. Confirm in Dashboard
```
Buffer.com → Dashboard
You should see the post in the "Scheduled" queue
```

## Platform-Specific Tips

### Facebook
- ✅ Longer captions work well (2-3 sentences)
- ✅ Include call-to-action (Book Now, Learn More)
- ✅ Hashtags: 3-5 max
- ❌ Avoid external links if possible (Buffer will add the blog URL)

### Instagram
- ✅ Use emoji (✨, 🏠, 💰, 📱)
- ✅ Hashtags: 10-20 (max 30)
- ✅ Line breaks for readability
- ❌ Keep text under 2,200 chars
- Pro tip: Instagram Shows → post behind-the-scenes or testimonials

### LinkedIn
- ✅ Professional tone (you are B2B here)
- ✅ Lead with the value prop
- ✅ Longer-form content performs well (3-5 paragraphs)
- ✅ Hashtags: 3-5 relevant to business/real estate
- ❌ Don't use too many emoji

### X (Twitter)
- ✅ Keep it punchy (thread format)
- ✅ Use relevant hashtags (#CashBuyer, #RealEstate, #MD, #VA, #DC)
- ✅ Include link to blog
- ❌ Max length still applies (280 chars + link)

### Pinterest
- ✅ Focus on the image (Buffer auto-pulls from blog)
- ✅ Pin description = caption
- ✅ Hashtags: 5-10
- ✅ Use keywords in description (sellers, cash home buyers, etc.)
- ✅ Re-pin old posts (top performers) monthly

## Tracking Performance

### In Buffer.com Dashboard
- **Click/Engagements:** How many people clicked the link
- **Impressions:** How many people saw the post
- **Clicks:** Direct link clicks to your blog
- **Engagement Rate:** Likes, comments, shares

### In Google Analytics (GA4)
- Go to GA4 → Traffic → Source/Medium
- Filter for "buffer.com" or "social" traffic
- Track which blog posts get most clicks from social

### Success Metrics (Goal)
- Week 1: 5-10 clicks to blog
- Month 1: 30-50 total clicks from social
- Month 3: 100+ monthly clicks from social

## Troubleshooting

### Post not showing in Buffer dashboard
- Refresh the page (F5)
- Check different tab: Scheduled vs. Published
- Verify the date/time is in the future

### Can't find the blog URL to paste
- Format: `https://www.parlevugloballlc.com/blog-{slug}.html`
- Example: `https://www.parlevugloballlc.com/blog-we-buy-houses-baltimore-fast.html`
- Copy from BUFFER_CAPTIONS.txt (URL is listed for each post)

### Schedule button is grayed out
- Make sure you selected at least one platform
- Make sure you entered a caption
- Make sure the scheduled date is in the future

### Post published but no traffic
- Buffer takes 30-60 min to send the post
- Most engagement happens in the first 2-4 hours
- Check GA4 → Social traffic to confirm clicks
